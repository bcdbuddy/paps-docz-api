## Obtenir les infos sur une tache donnée

Après avoir créé une tâche via l'API, vous pouvez vérifier qu'elle a bien été créée en utilisant la méthode `GET /api/v1/viewTask?id=<task-id>`.

| Paramètre | Type   | Description                                      |
| --------- | ------ | ------------------------------------------------ |
| `id`      | Number | L'unique identifiant de la tâche qui a été créée |

Cette requête est utilisée pour obtenir les détails de la tâche. Losrque tout se passe bien, vous aurez une réponse de ce type:

```json
{
	"message": "Successful",
	"status": 200,
	"data": {
		"fleet_id": 12201,
		"fleet_name": "Bassène",
		"fleet_latitude": 14.71723,
		"fleet_longitude": -17.16229,
		....
	}
}
```
