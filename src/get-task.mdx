## Obtenir les infos sur une tache donnée

Après avoir créé une tâche via l'API, vous pouvez vérifier qu'elle a bien été créée en utilisant la méthode `GET /api/v1/task?getId=<task-id>`. Exemple:

1) A travers l'identifiant de la tache

Cette api est utilisée pour obtenir les détails de la tâche. 
On aura sur l'URL `GET /api/v1/task?getId=<task-id>`. Pour avoir les informations de la tâche `getId` plus le numero du jobId. 

Losrque tout se passe bien, vous aurez une réponse de ce type:

```json
 {
	"payload": {
		"message": "Successful",
		"status": 200,
		"data": [
			{
				"job_details_by_fleet": null,
				"fleet_id": null,
				"fleet_name": null,
				"fleet_latitude": null,
				"fleet_longitude": null,
				....
			}
		]
	}
}
```
